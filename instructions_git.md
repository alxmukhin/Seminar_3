# **Инструкция по использованию системы контроля версий Git**

## Что такое Git?

Git это программа предназначенная для управления версиями проекта и отслеживания изменений в нем.

## Инициализация репозитория

Чтобы инициализировать (создать) новый репозиторий необходимо использовать команду:
    
    git init

## Проверка состояния репозитория

Чтобы проверить состояния репозитория необходимо использовать команду:

    git status
